** le dossier inputs contient les entrees necessaires pour executer le script : 
	-countries_code : ce fichier contient les codes des pays pour lesquels nous demandons les données.
	-liste_indicateurs : ce fichier contient la listes des fichiers excel contenants les données,
	    sous la forme => nom du ficher : titre variable .par exemple : 
	    API_NE.IMP.GNFS.ZS_DS2_fr_excel_v2_1002434:Importations de biens et de services (% du PIB)
	NB: le respet de cette forme est obligatoire!
 
** le dossier indicateurs contient les fichiers excel des indicateurs.
